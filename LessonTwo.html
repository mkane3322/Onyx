Lesson Two: Loops and Syntax

Loops

When it comes to coding you are bound to deal with repitition. Rather than writing something over and over again you can use what we call a loop. Loops allow you to set parameters that when fulfilled will complete the task until that parameter isn't fulfilled. There are many different types of loops that you will learn about in the coming sections. These types can be can applied through different situations based on what you want your code to do.

Each loop can have multiple parameters that can be connected through conditions that you set. These are common like 'or', 'and', '=', '+', and '-'. These will aid in specifying what you want your loop condition to be. These conditions will vary from language to language but will follow the same rules. Use these conditions as tools when dealing with your loops and their parameters.

While Loop

The first loop that we will be looking at is the while loop. This loop is the simplest to understand as it will run while the parameter is being fulfilled. The parameter will be put in parenthese after declaring the loop. There is a specific syntax associated with the parameters that you can set. You can include these as a connector as represented below.

The basic syntax of a while loop can be written as follows:

while (parameter) {}

For Loops

There will come a time where you want a loop to only run for a set number of iterations. The best loop to use for this circumstance is a for loop. This differs from the while loop as you will declare a variable for your iterations in the parameters. There are three main parts that are key when making a for loop.

The first part of a for loop is the declaration. For this most of the time you will declare the variable as 'i' and set it equal to 0 or some other number. This will play a part in what number you want your loop to start on. You can of course set a different variable name for the declaration, but since this variable will be declared within the loop 'i' is simple and effective.

The next part in the for loop is the condition. This will set the number of times that your loop will run. Genrally you would set your variable i as greater than '>' or lesser than '<' the amount of times you want your loop to run before it ends. This part of your loop will be seperated by a semicolon ';' as seen below when the syntax is broken down.

The final part in the parameters of a for loop is what you want your loop to do after every iteration. This will typically affect what your current i count is after each iteration. This can be achieved in many different ways. The most common way would be either 'i++' or 'i--'. When the '++' operator is added to a variable you should see it as 'i + 1 = i' and vice versa for 'i--'. This will increase or decrease your iteration count after each run of your loop.

Now let's put it all together to give you a visual aid of the making of a for loop:

for (i = 0 (initialization); i < 0 (condition); i++ (iteration result)) {}

If Else Loops

The next loop type that we will be looking at we have if else loops. These will also be commonly used throughout your coding journey. The if else loops is comprised of 2 main componants being your if condition and your else condition. The first part being your if condition. This is the section that will contain a parameter. The parameters follows the same rule that you've seen in the previous sections. After your parameter you will add brackets '{}' and then write your code. This is followed by your else function which doesn't have parameters but does include bracket and code when your if condition is not met.

You may also have an else if statement after your original if statement. These will have parametrs and brackets, and can be repeated but each specific condition you want a solution for. The syntax for all of an if else loop will be contained below:

if (parameter) {} else {}

if (parameter) {} else if {} ..... else if {}

Switch Statement

Next we will be looking into the switch statement which is not your traditional loop but allows for multiple cases to be played out based on the conditions you set. They are used when dealing with multiple conditions based on the same variable. You would first declare your switch statement with your condition in parantheses '()' followed by brackets '{}'. Inside those brackets is where you will begin to set up your cases. After you set up your case you have to put a break statement so the code can continue on since the case was met. You can have as many cases as you want and they will follow the same syntax:

case #: statement
break;

So if we put it all together:
switch (condition) {
case #: statement
break;
}
